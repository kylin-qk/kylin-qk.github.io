指针类型转换

void fun(char);
int a=125,b;
fun((char)&a);
void fun(chars)

charc;
c=(s+3);(s+3)=(s+0);(s+0)=c;
c=(s+2);(s+2)=(s+1);*(s+1)=c;
}
注意这是一个 32 位程序，故 int 类型占了四个字节，char 类型占一个字节。函数 fun 的作用是把一个整数的四个字节的顺序来个颠倒。注意
到了吗？在函数调用语句中，实参&a 的结果是一个指针，它的类型是
int *，它指向的类型是 int。形参这个指针的类型是 char *，它指向
的类型是 char。这样，在实参和形参的结合过程中，我们必须进行一
次从 int *类型到 char *类型的转换。结合这个例子，我们可以这样来
想象编译器进行转换的过程：编译器先构造一个临时指针 char *temp，
然后执行 temp=(char *)&a，最后再把 temp 的值传递给 s。所以最后的结果是：s 的类型是 char *,它指向的类型是 char，它指向的地址就是a的首地址

该书的微妙之处在于，它会让你知道你应该让计算机算什么。相比之下，如何计算显得微不足道，因为计算机远比你算得好。

第三章-微分中值定理与导数的应用

微分中值定理

罗尔中值定理

闭区间连续，开区间可导，区间端点值相等，则至少存在一点&，使得f(&)的导数=0；

拉格朗日中定理

闭区间连续，开区间可导，则至少存在一点&，使得切线斜率等于割线斜率；做辅助函数的思想；
推论，如果函数f(x)再开区间I内的导数恒为零，那么f(x)在区间I内是一个常数；

柯西中值定理

闭区间连续，开区间可导，分母导数（一个）不为0，那么再开区间上至少存在一点&，使得两函数的该点导数之比等于两函数区间端点值之差之比

洛必达法则

记忆等价无穷小，及几种指导手册上的几种类型

泰勒公式

记住p104几种常见的泰勒展开式；在计算中进行替换；

函数的单调性 极值与最值

知道驻点是导数等于0的点；

函数的凹凸性，渐近线及图像描绘

1，若函数f(x)在区间I上连续，在I0内有二阶导数，那么若二阶导数>0,则f(x)的图像在I上是凹的；
2，左右凹凸性相反的点曲线的一个拐点；
3，曲线的渐近线，求出k和b的值即可；

曲率

1，弧微分（p123公式)；
2，曲率及计算公式；圆曲率等于圆半径的倒数，圆的半径越小，曲率越大，弯曲的越厉害；
3工程上常用曲率圆在一点处的一段圆弧来近似替代小曲线弧。

定积分的概念和性质

  • 定积分中还用到了导数和极限；
  • 定积分是个常数，具有保序性；
  • 设M及m分别是函数f(x)在[a,b]上的最大值及最小值，则估计积分范围的方法为 积分在m(b-a)和M(b-a)之间；
  • 积分中值定理，如果函数f(x)在闭区间[a,b]上连续，则在[a,b]上至少存在一点t,使得其定积分等于f(t)(b-a) (t在a到b 之间);

    2，原函数存在定理一方面表明了连续函数的原函数是存在的，另一方面揭示了积分学中原函数与定积分两者之间的关系；

#指针小结
1，指针是一个变量，只不过存储的是另一个变量的内存地址；
2，指针变量可以赋值，指针的指向在程序执行中可以改变；
3，数组储存在一块连续的内存空间中，数组名就是 这块连续空间的首地址；
4，为指针数组的指针赋值：int *ptr num可以把数组名直接赋值给某个指针；也可以int *ptr num=&num[0]，数组首元素赋值给指针
