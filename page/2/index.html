超级推荐

发表于 2020-04-26

超级推荐的相关知识，快来理解一波吧
原创 业余的小忙人 零边际 1周前

超级推荐流量优化策略
超级推荐的核心原理：
超级推荐的流量趋势为推荐式流量，展现的方式包括：在猜你喜欢的首页/购中/购后中展现；在微淘中展现；在直播广场/特价专区展现；如果产品有入池，用超级推荐投放时，可以拿到大量的首页流量或者给产品打上标签；超级推荐的出价原理：很多卖家会遇到这么一个问题，在调整价格以后，计划的流量会逐渐流失。这时就先了解不同工具的区别：直通车是竞争机制，是与人的竞争，竞争的是排名；超级推荐是机器算法，是系统匹配的，比如之前出价为0.6，因为计划有收益，加价至1，这时，系统就会误以为之前的池子有误，就会重新匹配，这时缓冲的匹配时间就会带来匹配的误差，因此这就是加价出现的很大的误区，因此卖家切记不能频繁切换价格，卡着一定的价格出，发现效果好加预算就可以了，无须加预算。
超级推荐流量的诊断优化策略
任何付费推广工具都需要计算盈亏平衡点：1/毛利率，然后分析数据，转化周期选择30天，看投入产出比，如果投入产出比＜盈亏平衡点，则处于亏损状态。在商品推广中，主要看投资回报率；在图文推广里边无法看投资回报率，更多看拉新成本和投资成本；加购成本=花费/加购，收藏加购率=（收藏+加购）/点击量，将总的成交金额-商品推广成交金额，就能知道图文推广和直播推广带来的成交金额，对比相应花费，就能知道投产比。
此外超级推荐中，往往都是多计划一起进行投放的，对于投资回报率差的计划不能一刀切，因为这样会很可能切断店铺的部分流量，尤其新品期，一刀切会导致流量的全部流失。同样根据上面的表格模板算出表格的各项指标，从成本的角度，筛选出好的计划。在资源位中，一般投放都会好几个资源位投放，这时需要单独做一个计划，把溢价比例加到10%-30%；在创意中，停掉花钱特别多切效果特别差的创意，同时查看点击率，留高点击率的创意，把低点击率的创意移除，在创意开始跑的时候，放2张创意图就可以了，因为图片过多会拉低点击率。如果是打标签的计划，只做关键词定向就可以了，相似同行人群、达摩盘的人群就不需要投放了，因为同行的人群数据可能会有水分，当然如果能够确定行业数据没有水分，用这些人群效果是很好的。新品打标签的时候，老款就不需要打标了，只能修正标签，新品打标签中有一个新品推广，重点选择促进目标/促进点击率，且选择自定义计划的拉新定向中的关键词定向，切记这时必须关闭智能定向，因为智能定向会根据店铺人群来圈，这时店铺人群不精准就白费力气了；老款如果人群不精准，就需要修正标签，首先关闭智能定向，然后用关键词定向或者用达摩盘标签人群去圈，因为有一定流量的店铺层级都不会太差，达摩盘的精准度也不会太差。
答疑收录

  1. 展现量很大，但点击率很差，我加价效果更差怎么办？
    这时不能出价，问题出在图上，要换图。
  2. 怎么判断店铺人群标签精不精准？
    生意参谋—流量—访客分析—日期选择30天—看新买家、老买家、未支付买家的特征占比，如果新的流量与支付买家人群标签一致，即比例图比较类似，拉的标签就很精准。
  3. 不准的类目怎么办？
    继续测，测到准的类目为止
  4. 付费关掉了，免费流量也没有了怎么办？
    就是免费流量没有爆起来，需要重新开通付费
  5. 产品类目限制，不能推荐手淘首页怎么办？
    去超级推荐里面看看能不能拿到好的流量和数据，如果可以，可以去投放。
  6. 超级推荐和直通车哪个效果好？
    看店铺哪个渠道有优势。

直通车的技巧

发表于 2020-04-26

​直通车测款选款技巧
直通车测款流程
推广设置测试计划产品设置关键词->获取流量测试出价获取流量积累数据->数据分析产品数据化分析横向产品对比产品确认
测款的方法有很多，钻展，淘宝客，免费流量等，但点击率只有直通车才有，数据最准确。
测试产品数据的分析
高点击率，高收藏加购率，高访客价值——爆款产品；
高点击率，高收藏加购率，低访客价值——潜力款产品；
高点击率，低收藏加购率，低访客价值——引流产品；
低点击率，高收藏加购率，高访客价值——利润款产品；
挖掘店铺潜力产品技巧
1.产品决定店铺盈亏
(1)背景:在竞争环境中，越来越多卖家遇到了这个危机:流量大幅下滑，访客急剧下降，直通车等付费渠道的投入产出比也不高，根本原因就是产品的质量好坏，而产品动销率逐渐下降，正表明产品是负向反馈的 。
(2)产品负向反馈特征
特征1:免费流量获取能力差 特征2:推广亏钱严重 特征3:没有爆发力
(3)那么如何寻找好的产品呢？
首先不要主观认款，即使选款能力非常强，依然需要不断的去做一些数据的分析，即要多做测款的动作，不要做认款的动作；如果卖家本身是工厂，要注意筛选好范围，先测好款，再组织产品生产，注意要让产品多样化，可以分散风险。
发掘一款值得推广的产品
首先直通车选款测款的方法为产品带来流量，把直通车上的优质词加入，把劣质词删除；要记住产品是核心竞争力，产品决定生命周期，产品决定店铺的盈利能力。花费越高的费用，投产比越低，如果想做大流量，投产比会低一点。
公平公正数据化选款
收藏加购率，访客平均价值，点击率是非常关键的三个指标。以女装为例思考问题一，女装春节之后什么产品最赚钱？一个是小西装，一个是牛仔外套，此外一个子类目下最多两个爆款产品，你在推动新产品只会把你原有的爆款产品的流量给的吸掉。
踩准节奏跟紧大盘很重要，首先要了解整个市场行业动向和行业数据，能给到我们非常多的指导，并进行数据洞察和分析抓住机会类目流量。
那么这个类目什么时候有机会，可以打开生意参谋中—市场—搜索排行，搜索排行中，大家再来选择顶部的子类目，比如我是女装，我就选女装女士精品，查看最近一段时间的飙升词和搜索热词，既飙升又热搜的产品自然可以做。
分析产品的数据参考指标：点击率、加购率、收藏率、转化率和访客平均价值，那么如何分析呢？首先打开品类-选择最近7天-下载数据表，对数据表做以下整理，计算收藏加购率=（商品购物车件数+收藏的人数）/访客数，看整体收藏加购率最高的产品，一般女装到达14%-15%才算爆款；访客平均价值=该商品销售额/该商品的访客数，极限引流成本=访客平均价值*毛利率，最高支持引流的情况，极限引流成本是非常重要的衡量指标，相同毛利率，访客价值越高，商品便会越好。
总结归纳
常见的疑问
问题1:数据化运营现在都做精细化运营吗？
是的，直通车，超级推荐这些都是技术，但要把握细节，才能发掘数据背后的价值。
问题2:必须要有多少个访客有参考价值？
500个才算精准流量，至少有500个数据才能做分析。
问题3:飙升词过段时间下降怎么办？
一般飙升词不会是短时间的，都是淘宝上万个消费者搜索才会出现的，因此不会短时间的。
问题4:收藏加购每天都有，但没有转换？
不要看收藏加购数，要看收藏加购率，加购率高才算好产品。
问题5:500访客是周还是天啊？
不用看周或天，时间没有关系累计就行。
问题6:上白底图说涉及宗教怎么办？
问下阿里妈妈客服咨询一下。

手淘首页玩法 一

图的重要性及操作

手淘首页的流量的特点是：流量大，转化率低，数据好看，点击率高但是人群不稳定。核心要素：1，点击率；2，收藏加购率，判断人气值是否达标（注：单品收藏，收藏加购率=加购量/访客量）保持在同行之上；决定你点击率的是主图，这是第一大核心要素。单品的五张主图一定要加满，其中第五张主图一定要白底，也可以加场景图：无文字，五水印，五logo边框；其他操作进入银河素材中心鹿班设计中进行主图优化。另外注意在猜你喜欢，有好货里面的宝贝不是自己上传的，而是系统选择的！如果不会做白底图，这里附加一个小小的网站，自动抠图（https://www.gaoding.com/clip?id=57301366),并且无水印，具体功能自己尝试就清楚了。要是想加文案的话，可以在鹿班智能设计里面操作**智能文案**模块自己添加；一切操作完成之后，系统会自动审核，但审核的周期会比较长，超过一个星期还没审核通过的话建议重做，通过后自动入池。

需了解

原创图片，有特点的店铺，定位统一的店铺更容易如池，至于店铺介绍，可以参考其他同行。

手淘猜你喜欢测图

用这种方法提高点击率，进入入口方式：卖家中心-店铺管理-智能设计-鹿班设计-主图点击优化。每个商品上传四张不同的场景图，进行投放，起码根据一周的效果进行分析，是否换图！

重点总结

1，4大核心权重依次排序，必须先满足点击率超过行业至少1.5倍以上，才能获得猜你喜欢的展现；2，产品本身的竞争力强，加购收藏率也必须超过行业或者直接竞争对手的1.5倍，首页流量才会增长；3，猜你喜欢的坑产持续，并且展现价值高于对手，首页流量才会持续稳定的增长。

c++

  1. 相同的函数名可以有若干个不同的函数体，在调用同一函数时由于不同的环境可能有不同的行为，这种功能叫做函数的多态
  2. 类的一般定义形式class<类名>{私有，保护，公用}；
  3. 成员函数定义的结构如下：<类型><类名>::<成员函数名>(<参数表>)；类的成员函数是对累的私有数据成员进行操做的唯一途径。
  4. 构造函数也是类的一个成员函数，构造函数的函数名与类名相同，而且不能有任何返回类型，也不能标为void
  5. 构造函数定义对象的语法结构为：<类名><对象名>（<构造函数的参数表>）；
  6. 构造函数在定义对象时被直接调用，程序中不能直接调用构造函数
