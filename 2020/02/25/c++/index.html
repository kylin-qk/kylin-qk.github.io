c++

  1. 相同的函数名可以有若干个不同的函数体，在调用同一函数时由于不同的环境可能有不同的行为，这种功能叫做函数的多态
  2. 类的一般定义形式class<类名>{私有，保护，公用}；
  3. 成员函数定义的结构如下：<类型><类名>::<成员函数名>(<参数表>)；类的成员函数是对累的私有数据成员进行操做的唯一途径。
  4. 构造函数也是类的一个成员函数，构造函数的函数名与类名相同，而且不能有任何返回类型，也不能标为void
  5. 构造函数定义对象的语法结构为：<类名><对象名>（<构造函数的参数表>）；
  6. 构造函数在定义对象时被直接调用，程序中不能直接调用构造函数
